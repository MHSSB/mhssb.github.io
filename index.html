Features

Feature #1

This feature allows you to get information about a Minehut server. Let's use the server Golf as an example. All you have to do is /server (name), then it'll give you the information from that server.

Feature #2

Our next feature that we can showcase to you is "/icon". This allows you to get the icon of a Minehut server. This only gives you either a custom icon or the default Minehut icon. The custom "item" icons, however, aren't supported for this bot and might be added in the future.

Feature #3
